Welcome to the ScanOMetric documentation !
==========================================

ScanOMetric is a Python library for automated morphometric analysis of MRI anatomical scans.

Check out the :doc:`usage` section for further information, including
how to :ref:`install <installation>` our app.

.. note::How to cite

   Rummel, C., Slavova, N., Seiler, A. et al. Personalized structural image analysis in patients with temporal lobe epilepsy. Sci Rep 7, 10883 (2017). https://doi.org/10.1038/s41598-017-10707-1
